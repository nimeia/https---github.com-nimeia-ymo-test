# 错题重练队列 + 同类再出一题

本轮补齐了“专题闭环”这一层，目标是把原先的单题闭环推进成：

1. 当前题做错
2. 可加入错题重练队列
3. 可立即触发“同类再出一题”
4. 可在结算页 / 错题本统一启动队列重练
5. 可启动“队列 + 同类再出一题”的专题闭环练习

## 已完成能力

- 错题重练队列
  - 错题可手动加入/移出队列
  - 队列持久化到 localStorage
  - 首页、做题页、结算页、错题本都可启动队列练习

- 同类再出一题
  - 按同子题型优先、同模块次优、同交互和难度接近加权排序
  - 做题页与错题本均可从当前题触发同类加练

- 专题闭环入口
  - 队列重练：仅重练当前错题队列
  - 队列专题闭环：为队列中每道题自动补一题同类加练题

## 关键文件

- `h5-mvp-react/src/lib/topicClosure.ts`
  - 错题队列结构
  - 同类题排序
  - 队列专题闭环题目生成

- `h5-mvp-react/src/App.tsx`
  - 首页、做题页、结算页、错题本的闭环交互入口
  - 队列状态持久化与路由切换

- `h5-mvp-react/src/styles.css`
  - 队列卡片、同类推荐、错题本操作区样式

## 当前策略

- “同类再出一题”默认开启一个 1 题微练 session
- “队列专题闭环”会按 `错题 -> 同类加练题` 的顺序展开一个新的 session
- 当前版本不自动在答对后移出队列，保持教师 / 家长可见和可控
